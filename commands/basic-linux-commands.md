## Basic Linux Commands

ls - list directory contents - lists files and directories in the current directory

cd - change directory - cd /path/to/directory - changes the current directory to the specified path

pwd - shows working directory - displays the full path of current directory

## Files and directories

mkdir - create a new directory - mkdir cybersecurity - creates a directory named cybersecurity

rmdir - remove a directory - rmdir cybersecurity - removes a directory named cybersecurity

rm -r - remove a directory with it content - rm -r cybersecurity

rm - removes files or directories - rm file1.txt - deletes file1.txt

touch - create a new file - touch file1 - create a new file named file1

echo - print text to a terminal or to a file - echo "Today is the day" > file.txt - write a text into a file.txt, if a file doesnt exist, command creates one

cat - concatenate and display file contents - cat file.txt - shows contents of file.txt

cp - copy files or directories - cp file1 file2 - this command copies file1 to file2

mv - moves or renames files or directories - mv file1 file2 - moves or renames file1 to file2

## Screen editing

clear - clears the terminal screen

exit - exit the shell or current session
